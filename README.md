## 🛠️ What I am Learning Right Now
* 🧠 **Core Tech:** I am actively learning **Embedded C** and the fundamentals of **8-bit AVR microcontrollers**.
* 🐧 **Linux OS:** I am deliberately developing on a **Linux system** to learn the OS inside out, understand its build tools, and master the command-line environment—crucial skills for modern embedded engineering.
* 💻 **Bare-Metal Basics:** To understand what actually happens inside the chip, I avoid high-level frameworks. I use an Atnel ATB development board and standard Arduino hardware, but I program them **completely without the Arduino IDE wrappers**, writing directly to registers.
* 🕹️ **Current Sandbox Project:** I am building a simple [Catch the LED Game](https://github.com/LukaszMateuszSobczak/catch_the_led_game). It is a basic project used purely to practice register manipulation, state management, and moving away from blocking delays.
